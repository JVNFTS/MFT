Genshin Impact

Genshin Impact is a 2020 action role-playing game produced by MiHoYo/HoYoverse. The game features an anime-style open world environment and an action-based battle system using elemental magic and character-switching. A free-to-play game monetized through gacha game mechanics, Genshin Impact is updated regularly using the games as a service model; it was originally released for Android, iOS, PlayStation 4, and Windows, followed by PlayStation 5 in 2021, with an Xbox Series X/S version in November 2024.


Alasan :

  • Visual bergaya anime dengan grafik dan pemandangan yang indah
  • Game ini free-to-play dengan kualitas setara game AAA
  • Cerita dan lore yang kompleks dan menarik
  • Bisa dimainkan di PC, mobile, PS4/PS5 dengan fitur cross-save
